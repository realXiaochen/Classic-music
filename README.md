# Classic music (python/Django)  

- Video streaming site deployed on Heroku.com
- Demonstration of Django skill
   
## Tool Used
- Django
- Crispy form
- Registration
- Bootstrap

## Features
- Registration, login/out
- Video steaming
- Search by name, category, composer
- Related video
- Pictures and videos copied from Youtube
